# Protoblocks
Website for Protoblocks.  
Base project taken from [react-redux-universal-hot-example](https://github.com/erikras/react-redux-universal-hot-example)  

## Installation

```bash
npm install
```

## Running Dev Server

```bash
npm run dev
```

## Building and Running Production Server

```bash
npm run build
npm run start
```
